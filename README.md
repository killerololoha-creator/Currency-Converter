## Описание программы
Графическое приложение для конвертации валют с использованием актуальных курсов из внешнего API. Поддерживает 12 популярных валют, сохраняет историю конвертаций и позволяет загружать её при запуске.

### Возможности:
- 💱 Конвертация между 12 валютами (USD, EUR, RUB, GBP и др.)
- 📊 Актуальные курсы через ExchangeRate API
- 📝 Сохранение истории конвертаций в JSON
- 🔄 Загрузка истории при запуске
- ✅ Валидация ввода (положительные числа)
- 🎨 Красивый GUI на Tkinter
- 🌐 Онлайн обновление курсов

## Получение API-ключа

1. Перейдите на сайт: https://app.exchangerate-api.com/sign-up
2. Зарегистрируйтесь (бесплатный план - 1500 запросов/месяц)
3. Подтвердите email
4. Скопируйте ваш API-ключ из личного кабинета
5. Вставьте ключ в переменную `API_KEY` в файле `currency_converter.py`
